Benchmark between Deno and Bun

REST API Performance comparation between both platforms, using Hono framework as the reference implementation in both cases Furthrr technical details: bench-deno-bun All tests are launched during 5 seconds, 2 workers and 200 connections, the displayed results are the average data

Commands

  • Ping: Return a simple text
    • Example response: PONG
  • Version: Return a simple json with app version
    • Example response: 
      {
    "server": "H3lp3r API",
    "version": "1.0.4",
    "platform": {
        "bun": "v0.1.11"
    }
}
  • Echo: Return the same payload sent in the POST call
    • Example response: 
      {
    "foo": "Lorem impsup whatever",
    "success": true,
    "coefficient": 567.76,
    "page": 23,
    "list": [
        {
            "name": "Rob",
            "birdDate": "2000-02-03",
            "courses": 12
        },
        {
            "name": "John",
            "birdDate": "1990-10-13",
            "courses": 112
        },
        {
            "name": "Harry",
            "birdDate": "2002-12-05",
            "courses": 8
        }
    ]
    }
}
  • SHA512: Return json with a SHA512 hash
    • Example response: 
      {
    "result": "b1f4aaa6b51c19ffbe4b1b6fa107be09c8acafd7c768106a3faf475b1e27a940d3c075fda671eadf46c68f93d7eabcf604bcbf7055da0dc4eae6743607a2fc3f",
    "input": {
        "action": "sha512",
        "params": {
            "text": "Lorem ipsum dolor sit amet"
        }
    }
}
  • MD5: Return json with a MD5 hash
    • Example response: 
      {
    "result": "fea80f2db003d4ebc4536023814aa885",
    "input": {
        "action": "md5",
        "params": {
            "text": "Lorem ipsum dolor sit amet"
        }
    }
}
  • Base64 encode: Return json with a test encoded in Base64
    • Example response: 
      {
    "result": "TG9yZW0gaXBzdW0gZG9sb3Igc2l0IGFtZXQ=",
    "input": {
        "action": "encode",
        "params": {
            "text": "Lorem ipsum dolor sit amet"
        }
    }
}
  • Base64 decode: Return json with a plain test decoded from Base64
    • Example response: 
      {
    "result": "Lorem ipsum dolor sit amet",
    "input": {
        "action": "decode",
        "params": {
            "text": "Lorem ipsum dolor sit amet"
        }
    }
}
  • Random names: Return json with 100 generated person names
    • Example response: 
      {
    "result": [
      {
        "gender": "male",
        "firstName": "Hong",
        "lastName": "Óskarsdóttir"
      },
      { ... }
      {
        "gender": "female",
        "firstName": "Rut",
        "lastName": "Sigurjónsdóttir"
      }
    ],
    "input": {
      "action": "names",
      "params": {
        "total": "100",
        "gender": null
      }
    }
}
  • Ramdon email: Return json with a generated random email
    • Example response: 
      {
    "result": "toshioshapiro@example.net",
    "input": {
        "action": "email",
        "params": {}
    }
}
